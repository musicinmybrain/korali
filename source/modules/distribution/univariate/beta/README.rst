*******************************

Beta
-----
          
The Beta distribution has the probability density function (PDF):

:math:`f(x;k,\theta) =  \frac{x^{\alpha-1}(1-x)^{\beta-1}  }{B(\alpha, \beta)}`

where :math:`\alpha` and :math:`\beta` are the **shape** paramers.
