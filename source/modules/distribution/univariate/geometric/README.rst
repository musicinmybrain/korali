Geometric
---------

The Geometric distribution has the probability density function (PDF):

.. math::
    
    f(k;p)=(1-p)^k p \quad \text{for} 0\leq p \leq 1, k\in\{0,1,2,..\}

where :math:`p` is the **failure probability** and :math:`k` the number of **failures**.
