Poisson
-------

The Poisson distribution has the discrete probability density function (PDF):

.. math::

    f(x;\mu) = \frac{\mu^k}{k!} exp(-\mu)

where :math:`x \in \mathbb{N}_0` and :math:`\mu > 0` is the **mean** and the **variance**.
