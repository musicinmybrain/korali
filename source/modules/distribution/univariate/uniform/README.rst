Uniform
---------

The Uniform distribution has the probability density function (PDF):

.. math::

    f(x;a,b)=\frac{1}{b-a} \quad \text{for} b>a, x \in [a,b]

where :math:`b` is the **upper bound** and :math:`a` the **lower bound**.
