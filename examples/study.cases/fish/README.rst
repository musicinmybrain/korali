Study Case: Reinforcement Learning on Fish Swimming
=======================================================

In this study case, we replicate the fish swimming experiment from CSELab.

Setup
---------------------------

1) Install CubismUP2D by running:

.. code-block:: bash

   ./install_deps.sh

2) Compile the study case by running:

.. code-block:: bash
   
  ./make -j6

3) Run the test:

.. code-block:: bash
   
  ./run_test.sh
